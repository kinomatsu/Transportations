# 🚚 Перевозки

Система автоматизации деятельности транспортной компании, разработанная на платформе **.NET 8 (Windows Forms)** с использованием **Microsoft Access** в качестве базы данных.

Проект предназначен для учёта клиентов, водителей, транспортных средств, грузов и рейсов, а также формирования отчётности по перевозкам.

---

## 📋 Возможности системы

### Работа с клиентами

* просмотр списка клиентов;
* добавление новых клиентов;
* редактирование информации;
* удаление записей;
* поиск по данным.

### Работа с водителями

* учёт персональных данных;
* хранение информации о водительских удостоверениях;
* контроль статуса водителей.

### Работа с транспортом

* хранение информации о транспортных средствах;
* учёт грузоподъёмности;
* контроль текущего статуса транспорта.

### Работа с грузами

* регистрация грузов;
* хранение информации о массе и типе груза;
* поиск и редактирование записей.

### Работа с рейсами

* создание новых рейсов;
* назначение водителей и транспорта;
* контроль статуса выполнения;
* хранение стоимости перевозки.

### Отчёты

* формирование аналитических отчётов;
* просмотр статистики;
* экспорт данных в CSV.

### Безопасность

* авторизация пользователей;
* разграничение прав доступа;
* хранение паролей в виде SHA-256 хешей;
* защита от SQL-инъекций;
* защита от перебора паролей.

## 🏗 Архитектура проекта

Проект построен по принципу трёхслойной архитектуры.

```text
UI (Windows Forms)
        │
        ▼
BLL (Business Logic Layer)
        │
        ▼
DAL (Data Access Layer)
        │
        ▼
Microsoft Access Database
```

### Структура решения

```text
Transportations
│
├── Forms
│   ├── LoginForm
│   ├── RegisterForm
│   ├── MainMenu
│   ├── DatabaseForm
│   ├── ProfileForm
│   └── ReportsForm
│
├── BLL
│   ├── AuthService
│   ├── DataService
│   ├── PasswordHasher
│   └── UserSession
│
├── DAL
│   ├── DataRepository
│   └── DbManager
│   └── lDbManager
│
└── Tests
```
